Abstract

ABC is an interactive system for climbing, it was developed to create a customized experience for intermediate level climbers and let them develop their skills in the most effective way. Thanks to RGB leds and capacitive sensors embedded in the holds, ABC is able to generate new paths, collect and analyze data and generate suggestions and feedbacks for the climber.

We collected qualitative and quantitative data via surveys, inteviews, observation, shadowing and on field training. We then tried to convey both climbers and gym owners needs. The first wanted and needed more hints when got stucked, while the second wished a way to change the paths in an easier and faster way than doing it manually hold by hold.

Project selected to be presented in Milano Design week.

Design x Designers 2019

Problem setting

Climbing is one of the hardest sports because it needs something named “proprioception” that is the sense of self-moving and body position.

For this beginning and keep climbing becomes harder and harder, reason why most of the products or the initial climbing path are market with suggestions, color, or lights. A mix between digital and analog solution.

So, this project starts by asking what the best training is to learn to climb properly. All of them agreed that you learn climbing by climbing. This became our focus point that guided us through the entire project.

Brief

Focusing on indoor climbing, develop an interactive system in order to improve an intermediate climber body position and grabbing of the hand by a customizable path trought physical holds and augmented experience.

Product architecture

Design

The final design is a module-based system set. First, to achieve more versatility was necessary to create a changing tool with the capability of changing the hold shape. To get this, we designed a set of holds that contain different main shapes of climbing holds (Crimps, slopers, jugs, etc.). In order to show or highlight the path each of this hold contains a RGD LEDs sub-division embedded in a transparent PU resin. So, like this the climber can choose to train a specific set of skills or ability.

Embedded in the hold there are different parts, from capacitive sensor used to detect the climber position and holding time, to a 360 connection between the hold and wall, this to allow all possible degrees of rotation, versatility to change holds and providing data transfer and power supply.

The hold connection is a normal bold with a T-nut. The 360° ring that is attached to the wall, this allows the use of the NFC that is needed to recognize where each hold is located.

Finally, the ABC climbing app that helps to the climber to improve its experience, by having an interface to select the climbing path, collecting data and performance.
