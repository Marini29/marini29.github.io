DMX Controller

DMX Controller lights is a program developed in C#, the main scope of this projects was the integration between a lighting system and the UI/UX software. Like this, you can program the activation and the sequence of the DMX lights according with the needs of the event. Also, it was possible include sensor system like ultrasonic sensor to detect the user activity and deploy the DMX sequence.
